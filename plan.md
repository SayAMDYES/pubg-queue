# 前台报名页面
1. 点击已经报名的用户，弹窗打开用户战绩（参考战绩查询）
2. 已报名时就不要再显示报名按钮了
3. 退出报名按钮直接合并到已报名队伍自己的那个用户名后方的按钮里
4. 一键邀请无效，排查原因
5. 如果时间已经超过报名截止时间，报名按钮就不要显示了，也不可以退出

# 后台报名详情
1. 自动拉取战绩而不是点刷新战绩
2. 报名记录的“状态”怎么是“assigned”
3. 详情页可以快捷触发开始记录实际开始时间，结束时间也一样,而不是还要去点编辑
4. 当点击结束以后，拉取战绩，根据实际开始时间和实际结束时间去过滤所有比赛，然后自动计算出战绩排名并存储到数据库里，然后增加一个重新计算按钮
